# CSUST-Course-Resource-Download-Script.
一个可视化批量下载长沙理工大学网络教学平台的脚本
**安卓浏览器（X 浏览器、Via）和 PC 浏览器（Edge / Chrome + Tampermonkey）导入本地脚本 xxx.js** 的教程
## 一、安卓端：通过 X 浏览器或 Via 浏览器导入本地脚本
### 前提准备
1. 脚本文件 `xxx.js` 已通过 **QQ 或微信** 接收。
2. 在 QQ/微信中点击该文件 → 选择 **“保存到本地”**（或“用其他应用打开” → “保存”） 。
3. 文件通常保存路径为：
   - QQ：`/storage/emulated/0/Download/QQ/xxx.js`
   - 微信：  `/storage/emulated/0/Download/WeiXin/xxx.js`
     （部分机型可能在 `/storage/emulated/0/Android/media/com.tencent.mm/MicroMsg/Download/xxx.js`）

> ⚠️ 若无法看到完整路径，可使用系统“文件管理”App，在“内部存储/Download”目录下查找。


### 方法 1：X 浏览器导入本地脚本

X 浏览器内置脚本引擎，支持直接导入 `.js` 文件 。

#### 操作步骤：
1. 打开 **X 浏览器**。
2. 点击右下角 **“≡”** → 点击齿轮图标**“设置”**。
3. 找到并点击 **“用户脚本”**（或“脚本管理”）。
4. 点击右上角 **“新增”**。
5. 选择 **“从文件导入”**。
6. 在文件选择器中，导航至上述保存路径（如 `/Download/QQ/`），选中 `xxx.js`。
7. 系统会自动读取脚本元信息（如 `@name`, `@match`），确认后点击 **“保存”** 即可启用 。

## 二、PC 端：Chrome / Edge + Tampermonkey 导入本地脚本

### 前提
- 已安装 **Tampermonkey 插件**。
安装地址https://www.tampermonkey.net/index.php?locale=zh
- 脚本文件 `xxx.js` 已保存在电脑本地（如桌面或下载文件夹）。

### 导入步骤：
1. 打开浏览器，点击 **Tampermonkey 图标**（猴头图标）→ 选择 **“仪表盘”**。
2. 在左侧菜单点击 **“实用工具”**。
3. 找到 **“导入”** 区域 → 点击 **“选择文件”**。
4. 从文件管理器中选择你的 `xxx.js` 文件 → 点击 **“打开”**。
5. Tampermonkey 会自动解析脚本头信息（`// ==UserScript==` 部分），确认无误后点击 **“安装”** 。

> ✅ 安装成功后，脚本将在匹配的网站（如 `https://course.xxx.edu.cn`）自动生效。

## 三、使用场景说明（以校园教学平台为例）

假设你已成功导入用于下载课程资源的脚本（如 `campus-download.js`）：

### 使用流程：
0. 已开启脚本
1. 访问并登录网络教学平台
2. 进入某门课程
3. 点击 **“课程资源”**。
4. 此时页面右下角会出现 **脚本悬浮按钮**（“资源脚本”）
5. 点击该按钮 → 弹出资源列表 → 选择要下载的文件 → 点击 **“下载”**。
   - 或：进入具体文件预览页（如 PDF/视频），页面顶部会出现 **“下载”按钮**，点击即可保存。

## 四、常见问题

| 问题                   | 解决方案                      |
| -------------------- | ------------------------- |
| 脚本未运行                | 检查篡改猴是否在脚本选项中已开启允许运行用户脚本                   |
| 安卓找不到文件              | 使用“文件管理”App 确认文件是否真实保存到本地 |
| Tampermonkey 不显示导入选项 | 直接将js文件拖入到篡改猴页面           |
| 微信/QQ 文件无法保存         | 右上角三个点，点击，更多，选保存到本地       |
